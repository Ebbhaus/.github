# Support

Looking for help with an Ebbhaus project? Here's where to go.

- **Bugs and feature requests**: Open an issue in the relevant repository using
  one of the issue templates.
- **Security issues**: Follow [`SECURITY.md`](./SECURITY.md). Do not file them
  as public issues.
- **Internal questions**: Ask in the appropriate team channel in Slack.

Please don't use issue trackers for general Q&A or usage questions — they're
for tracking actionable work.
